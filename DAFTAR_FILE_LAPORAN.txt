   - Buka file .txt dengan Notepad atau text editor
   - Select All (Ctrl+A) dan Copy (Ctrl+C)
   - Paste ke Microsoft Word
   - Format sesuai kebutuhan (font, spacing, dll)

2. UNTUK DIAGRAM:
   - Diagram sudah ter-render dalam format visual
   - Screenshot atau export diagram sebagai PNG
